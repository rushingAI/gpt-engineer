# 项目模板系统

这个目录包含用于快速启动项目的预配置模板。

## 可用模板

### 1. react-ts-shadcn
现代化的 React + TypeScript + shadcn/ui 模板。

**特性:**
- ⚡ Vite 构建工具
- 🎨 Tailwind CSS + shadcn/ui 组件库
- 🎬 framer-motion 动画库
- 📱 响应式设计
- 🎯 TypeScript 类型支持
- 🔧 预配置的开发工具

**适用场景:**
- Landing Pages
- SaaS 应用
- Dashboard
- 营销网站
- 管理后台

## 模板结构

每个模板包含：
- `template.json` - 模板元数据和配置
- `files/` - 模板文件目录
  - 完整的项目结构
  - 预安装的依赖配置
  - 基础组件库

## 使用方式

AI 会在模板基础上生成业务代码，而不是从零开始，这样可以：
1. 保证代码质量和一致性
2. 使用现代化的技术栈
3. 快速生成可运行的应用
4. 专注于业务逻辑实现
